Tally Definition Language (TDL)

At the heart of Tally, TDL forms the key link between the application and the platform layer. TDL provides the user with flexibility and power to extend the default capabilities of Tally and integrate with external applications. TDL provides capabilities for rapid development, rendering, data management and integration. The Tally Definition Language (TDL) along with the development suite Tally.Developer 9(TD 9) provides the strong technology backbone for developing solutions on Tally platform
